# Security Audit Skill

Comprehensive security assessment of systems and code.

## Audit Types

1. **Code Security Review**
   - Static analysis
   - Dependency scanning
   - Secret detection
   - Input validation

2. **Infrastructure Audit**
   - Cloud configuration
   - Network security
   - Access controls
   - Encryption status

3. **Threat Modeling**
   - STRIDE analysis
   - Attack trees
   - Data flow analysis
   - Trust boundaries

4. **Compliance Assessment**
   - GDPR check
   - CCPA check
   - SOC 2 readiness
   - Industry standards

## Output

- Security report with findings
- Risk ratings (Critical/High/Medium/Low)
- Remediation steps
- Compliance status
